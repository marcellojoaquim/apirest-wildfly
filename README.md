# Gerenciamento de Produtos
<p>CRUD API REST de produtos, desenvolvido em java e configurado para o servidor wildfly com 
persistencia no Banco de Dados PostgreSQL</p>

## Tecnologias: 
 - Java 8
 - Jakarta EE 8 / Java EE 8
 - Wildfly 23 
 - PostgreSQL
 - Maven

## Pré-requisitos
- JDK 8
- Wildfly 23
- Apache Maven
